# electron-start

> Hot-reload
> Less compilation
> Linting with standard
> Devtools with Devtron


## Install

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
